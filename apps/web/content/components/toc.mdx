# Table of Contents

The table of contents component helps users navigate easily by listing the main contents of a document.

## Usage

```tsx
import {
  TableOfContentsList,
  TableOfContentsTitle,
  TableOfContentsItem,
  TableOfContentsLink,
} from "@/ui/toc"
```

## Components

### TableOfContentsList

A container that wraps the table of contents items.

### TableOfContentsTitle

Displays the title of the table of contents.

### TableOfContentsItem

Represents an individual table of contents item.

### TableOfContentsLink

Provides a link for the table of contents item.
