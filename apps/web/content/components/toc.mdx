# Table of Contents

A list of the main contents of a document.

## Structure

The `Table of Contents` component consists of the following components:

- `TableOfContents` - Container for the table of contents.
- `TableOfContentsList` - Wrapper for table of contents items.
- `TableOfContentsTitle` - Displays the table of contents title.
- `TableOfContentsItem` - Individual table of contents entry.
- `TableOfContentsLink` - Navigation link for table of contents items.

When properly implemented, the component creates the following Semantic HTML structure:

```html
<aside>
  <nav>
    <p>On This Page</p>
    <ul>
      <li>
        <a href="#introduction">Introduction</a>
      </li>
      <li>
        <ul>
          <li>
            <a href="#features">Features</a>
          </li>
        </ul>
      </li>
    </ul>
  </nav>
</aside>
```

## Usage

```tsx
import {
  TableOfContents,
  TableOfContentsList,
  TableOfContentsTitle,
  TableOfContentsItem,
  TableOfContentsLink,
} from "@/ui/toc"

<TableOfContentsList>
  <TableOfContentsTitle>
    On This Page
  </TableOfContentsTitle>
  <TableOfContentsItem>
    <TableOfContentsLink href="#introduction">
      Introduction
    </TableOfContentsLink>
  </TableOfContentsItem>
  <TableOfContentsItem indent>
    <TableOfContentsList>
      <TableOfContentsLink href="#features">
        Features
      </TableOfContentsLink>
      <TableOfContentsItem indent>
        <TableOfContentsLink href="#customization">
          Customization
        </TableOfContentsLink>
      </TableOfContentsItem>
    </TableOfContentsList>
  </TableOfContentsItem>
  <TableOfContentsItem>
    <TableOfContentsLink href="#installation">
      Installation
    </TableOfContentsLink>
  </TableOfContentsItem>
</TableOfContentsList>
```

## Components

### TableOfContentsList

A container that wraps the table of contents items.

### TableOfContentsTitle

Displays the title of the table of contents.

### TableOfContentsItem

Represents an individual table of contents item.

### TableOfContentsLink

Provides a link for the table of contents item.
