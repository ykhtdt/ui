# Table of Contents

목차 컴포넌트는 문서의 주요 내용을 목록화하여 사용자가 쉽게 탐색할 수 있도록 도와줍니다.

## 사용법

```tsx
import {
  TableOfContentsList,
  TableOfContentsTitle,
  TableOfContentsItem,
  TableOfContentsLink,
} from "@/ui/toc"
```

## 예제는

<ExampleTOC01 />

## 컴포넌트

### TableOfContentsList

목차 항목들을 감싸는 컨테이너입니다.

### TableOfContentsTitle

목차의 제목을 표시합니다.

### TableOfContentsItem

개별 목차 항목을 나타냅니다.

### TableOfContentsLink

목차 항목의 링크를 제공합니다.
